# Politica de Privacidade

O BarberFlow trata dados como nome, telefone, email, CPF opcional e dados de agendamento para operar reservas em barbearias.

## Uso dos dados

- Criar e autenticar contas.
- Registrar e gerenciar agendamentos.
- Enviar emails de recuperacao, convite, confirmacao e lembrete.
- Manter historico operacional e auditoria de seguranca.

## Retencao

Dados de agenda podem ser mantidos para registro operacional da barbearia. Quando o cliente solicita exclusao, seus dados pessoais sao anonimizados.

## Direitos do titular

O cliente pode solicitar exclusao pelo endpoint `DELETE /api/auth/me` autenticado.
